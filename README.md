
# TP3.2 – Tests d'Intégration avec Docker et Testcontainers

## Objectif
Apprendre à écrire des tests d'intégration Java en utilisant Docker et Testcontainers avec Spring Boot.

## Exercice 1 – Testcontainers
Reprendre les exercices du TP3 partie 1 et remplacer les mocks par de vrais conteneurs Docker. Réécrire les tests avec Testcontainers et pousser les images sur Docker Hub.

## Exercice 2 – Application de gestion de tâches
- Cloner le projet : [task-manager](https://github.com/rengreen/task-manager)
- Analyser les tests existants et leurs limitations
- Réécrire les tests d'intégration avec Testcontainers
- Exécuter les tests et vérifier qu'ils passent

## Technologies utilisées
- Java 8+
- JUnit 5
- Testcontainers
- Docker
- Spring Boot
